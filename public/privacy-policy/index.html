Privacy Policy

Effective date: May 7, 2026

QScaleUp provides a Strategic Marketing Tool that helps users create, manage, and complete marketing project documents using Google Drive and Google Docs.

Information We Collect

The app may access basic Google account information required for sign-in and may request permission to create, copy, rename, open, and delete project-related files and folders in the user’s Google Drive.

Google User Data

The app uses Google APIs only to provide the core project-management functionality requested by the user, including creating project folders, copying project document templates, opening Google Docs, renaming project folders, and deleting project folders when confirmed by the user.

The app does not sell Google user data. The app does not use Google user data for advertising. The app does not transfer Google user data to third parties except as necessary to provide or improve the app’s functionality, comply with the law, or protect users.

Local Storage

The app stores project state, task progress, and remembered connection information in the user’s browser local storage so the user can continue work after closing and reopening the app.

Data Deletion

Users can delete projects inside the app. When confirmed, the app may move the related Google Drive project folder to trash and remove the project from local app state.

Security

We use Google OAuth for authorization. Users can revoke the app’s access at any time from their Google Account permissions page.
