# Development Guide for Strategic Marketing Bootcamp

## Step-by-Step Guidance

### 1. Setup
- Clone the repository:
  ```
  git clone <repository-url>
  ```
- Navigate to the project directory:
  ```
  cd strategic-marketing-bootcamp
  ```

### 2. Install Dependencies
- Run the following command to install all required packages:
  ```
  npm install
  ```

### 3. Build the Project
- Use the following command to compile the TypeScript files:
  ```
  npm run build
  ```

### 4. Run the Application
- Start the application using:
  ```
  npm start
  ```
- Open your browser and navigate to `http://localhost:3000` (or the specified port) to view the app.

### 5. Testing
- Ensure all features work as expected, including:
  - User authentication via Google OAuth 2.0
  - Project creation and management
  - Task management and completion tracking

### 6. Deployment
- Follow the deployment instructions provided in the `README.md` file to host the application on your preferred platform.

### Additional Notes
- Make sure to configure your Google API credentials for OAuth 2.0 and the Google Drive API.
- Regularly check for updates in dependencies and keep the project up to date.
